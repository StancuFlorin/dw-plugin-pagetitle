DokuWiki plugin Page Title
====================================

Macro to set the title of the page in metadata.


Syntax
------


    ~~Title: title of the page~~ 

    ~~ShortTitle: short title of the page~~

Each of metadata is stored with key `title` and `shottitle`, respectively.
